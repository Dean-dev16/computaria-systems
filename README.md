📦 Computaria Systems

A simple Python system to simulate material purchasing, production, and product sales.

🧠 About the project

This project was created to practice:

Conditional statements (if)

Loops (while)

Dictionaries

User input with input()

Basic system logic

⚙️ Features

The system has 3 main options:

🛒 1. Buy (Material purchase)

Displays available materials

Allows selecting a material

Lets the user choose a quantity

Checks if there is enough stock

🏭 2. Production

Allows converting materials into products

Updates stock (⚠️ depends on variables already defined in the code)

💰 3. Sale

Displays available products:

Notebook 1

Notebook 2

Allows selecting a product

Checks stock availability

Confirms the purchase

⚠️ Important notes

Some variables like stock and product_stock are not initialized, which may cause errors.

This is a simple educational project, not intended for real-world use.

It can be improved with:

Error handling (try/except)

Code organization using functions
